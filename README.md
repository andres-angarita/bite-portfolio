# Bite — UX Case Study

Portfolio case study documenting the end-to-end UX design process for **Bite**, a nutritional tracking web platform designed for users with low digital literacy.

**→ [View live portfolio](https://andresangarita.github.io/bite-portfolio)**

---

## Project overview

| | |
|---|---|
| **Role** | UX Designer / Researcher |
| **Duration** | 3 months |
| **Tools** | Figma · Maze |
| **Year** | 2025 |

## Process

1. **Empathize** — User interviews, personas, empathy maps
2. **Define** — Journey maps, site map, hypothesis statements
3. **Ideate** — Crazy 8s, storyboards, competitive audit, user flow
4. **Prototype** — Paper wireframes → lo-fi → hi-fi V1 → hi-fi V2
5. **Evaluate** — Unmoderated usability study via Maze (A/B comparison)

## Key outcomes

- **−55% misclick rate** on critical tasks between prototype A and B
- **100% task success rate** across both prototypes
- 8 real participants in the usability study
- Predictive attention heatmap generated using a custom behavioral analytics tool
